CSS样式表

  1. 内联样式：将样式定义在元素的style属性里面，没有重用性
  2. 内部样式表：将样式定义在head里面的style中，仅限当前文档重用
  3. 外部样式表：讲样式定义在一个单独.css文件里面，再在head中引入该文件。 实现数据与表现分离

CSS选择器

  • 元素选择器 h1
  • 类选择器 .className
  • 分类选择器 h1.className
  • 元素id选择器 #myid
  • 派生选择器 h1 span
  • 选择器分组 h1,h2,span
  • 伪类选择器
    • :link 未访问过的链接
    • :visited 访问过的链接
    • :hover 悬停
    • :active 激活
    • :focus 获取焦点

CSS单位

  • 尺寸： % in cm mm pt px em
  • 颜色 rgb(x,x,x) rgb(x%,x%,x%) #rrggbb #rgb
  • 尺寸属性： width height overflow
